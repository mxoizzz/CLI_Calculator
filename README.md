# 🧮 CLI Calculator

A menu-driven Command Line Interface (CLI) Calculator built using **Java** as part of my **Project Ascend** journey. This project focuses on strengthening Java fundamentals by implementing commonly used mathematical operations while practicing clean code, methods, loops, user input, and program structure.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧮 Modulo
- 🔢 Power
- √ Square Root
- 📏 Absolute Value
- 🔄 Round
- ⬆️ Ceil
- ⬇️ Floor
- ❗ Factorial
- 🔍 Even/Odd Checker
- 📈 Maximum of Two Numbers
- 📉 Minimum of Two Numbers
- 💯 Percentage Calculator

## ▶️ How to Run

### Clone the Repository

```bash
git clone https://github.com/mxoizzz/CLI_Calculator.git
```

### Navigate to the Project

```bash
cd CLI_Calculator
```

### Compile

```bash
javac Main.java
```

### Run

```bash
java Main
```
